# 键盘钢琴

一个为手机与电脑设计的水晶键帽钢琴网页。页面先显示界面，641 个 Salamander
Grand Piano V3 采样按需读取并在后台保存，因此不会再因一个超大的 HTML 文件而
长时间停留在白屏。

## 离线使用

首次联网打开后，页面会在后台准备约 280 MB 的完整钢琴音色。进度完成后，可在
没有网络时继续演奏。iPhone/iPad 可在 Safari 的分享菜单中选择“添加到主屏幕”。

网页的所有操作都在同一个 `index.html` 界面中完成；音频、离线缓存和识谱资源
作为浏览器内部资源按需加载，以兼顾首屏速度和音质。

钢琴采样的来源与许可见 [LICENSE-SAMPLES.md](LICENSE-SAMPLES.md)。
